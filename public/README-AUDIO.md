# Audio Files

## Required Audio File

**buzzer.mp3** - This file should be placed in the `public` folder. It will be played when:
- A player presses the buzzer
- The host sees someone buzz in

You can use any buzzer sound effect you prefer. Recommended:
- Short duration (0.5-1 second)
- Clear, distinct sound
- Not too loud

Example sources:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://pixabay.com/sound-effects/

Once you have the file, simply place it at: `public/buzzer.mp3`
